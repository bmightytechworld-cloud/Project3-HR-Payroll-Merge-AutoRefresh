## HR Payroll Data — Cleaning, Merge, Auto Refresh & Summary

## Client Request
Client had two separate sheets — employee information 
and monthly payroll records. Needed them merged, 
cleaned, calculated and automated for monthly updates.

## Problems Found
**Employee Sheet:**
- Duplicate employee records
- Inconsistent department names (hr, HR, FINANCE)
- Inconsistent location names (lagos, LAGOS, Lagos)
- Mixed date formats across Date Joined column
- Missing employee name

**Payroll Sheet:**
- Duplicate payroll entries
- Missing Bonus and Deductions values
- Missing and incorrect Net Pay calculations
- Inconsistent month names

## What I Fixed
- Removed all duplicates from both sheets
- Standardized all department and location names
- Unified date formats across all rows
- Handled all missing values
- Calculated correct Net Pay for every employee
- Merged both sheets using Employee ID
- Built Input Sheet for future monthly payroll entries
- Set up Auto Refresh — client just adds new data and refreshes
- Built Summary Dashboard showing total payroll, 
  highest earner, average salary and total employees

## Deliverables
- Sheet 1 — Raw Employee Info
- Sheet 2 — Raw Payroll Data
- Sheet 3 — Clean Employee Info
- Sheet 4 — Clean Payroll Data
- Sheet 5 — Merged Data
- Sheet 6 — Input Sheet
- Sheet 7 — Summary Dashboard
